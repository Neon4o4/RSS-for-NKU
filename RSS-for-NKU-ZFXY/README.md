=======================
由数院RSS改编而来

工作原理：
=======================
**.xml：分类保存新闻的数据，供RSS阅读器更新。

RssSync.py：在服务器端定时在周政网站抓取数据，发现官网有更新便立即更新xml文件。

latest：保存上次更新时每个分类最新一篇文章的链接。
